# NotASketch

NotASketch is a YSWS program where teenagers create unique, unconventional drawing apps. The goal is simple: build something weird, creative, and fun — anything but a basic paint clone.

## Why NotASketch?

Most drawing tools try to be practical. NotASketch flips that idea — participants are encouraged to experiment with strange interactions, unexpected behaviors, and playful mechanics. Think drawing with physics, delayed strokes, randomized brushes, or controls that fight back.

The point isn't perfection. It's expression.

## How it works

1. Build a drawing app with at least one unique interaction
2. Track your time using Hackatime (code) or Lapse (art/design)
3. Publish it online so anyone can play it in the browser
4. Submit through the form on the landing page

## Prizes

- **Tier 1** — Sticker sheet (1-2 hours, basic drawing functionality)
- **Tier 2** — Sticker sheet + LCD writing tablet (2-3 hours, at least one unique interaction)
- **Tier 3** — Sticker sheet + Steam Gift Card (3+ hours, published and polished)

Projects are reviewed based on functionality, creativity, and clarity of interaction.

## About this repo

This repository contains the landing page for NotASketch. It features an interactive etch-a-sketch that draws as you scroll, a sketch mode with keyboard controls, and sections explaining the program, prizes, and FAQ.

No frameworks. No build tools. Just HTML, CSS, and vanilla JS.
